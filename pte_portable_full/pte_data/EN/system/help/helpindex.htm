Periodic Table Explorer v1.8
Welcome to the PTE help files. If you have any questions regarding the use of this program or wish to suggest new features, then feel free to send me an email.
The Periodic Table contains lots of properties and information on the elements. Click here to see a list of properties and how to find them.
 
The Menus
File->Clear Closes all of the opened windows
File->Load Layout Loads a layout
File->Save Layout Saves the current layout of windows
File->Save Layout As Same as above but to a different file
File->Set as Default Saves the current layout as the "default"
  (this layout is loaded every time the program is opened)
File->Exit Exits the program
Edit->Settings Opens the user settings dialog
View->Toolbars->Layout Toggle the Layout Toolbar (see below)
  Toggle the Quick view Toolbar (see below)
Language Change the application language
The Toolbars
This is the Main toolbar. Each button opens a separate application window. See the menu to the left for more information on each application.
This is the Layout toolbar. Use it to manage saved layouts and customise the look of the application, as below:
(a) Load the layout selected in (b).
(b) Select from previously saved layouts
(c) Save the current layout.
This is the Quick view toolbar. Use it when you want to open a specific data page quickly.
(1) Order the elements alphabetical by name or ascending atomic number.
(2) Select an element here
(3) Open a specific data page for the element selected in (2); left to right...
Main properties page
More info page
Compounds page
Isotopes page
Reactions page
Production/Isolation page
Spectra page
(4) Syncronise opened windows to the element selected in (2).
(5) Syncronise opened windows to the element selected in the selected window.
