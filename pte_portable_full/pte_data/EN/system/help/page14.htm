Toolbox
The toolbox is split in to three sections;
Equations
The toolbox features information on some important equations that might be useful to those studying chemistry.
Constants
The toolbox features 330 fundamental physical constants; their values and the uncertainty (inaccuracy) of the value.
Click on a constant to see its value.
Search within the names of the constants or click the "show all" button to display all the available constants.
from left to right;
copy the value, as it appears, to the clipboard
copy the value only (no units), so that it can be pasted in to a calculator
copy the value, in HTML format, to the clipboard
Chemical formulae
The toolbox features over 6400 chemical formulae, along with their CAS value (where available).
Search within the name (and alternate names), formula or CAS value of the chemical. Select an entry from the list of compounds that match your search from the list, the formula and CAS ID will be display beneath the list. To search for elements that contain a specific element use the following syntax; {symbol};
To find all elements that contain gold; {Au}
The molar mass, number of atoms of each element (within the compound) and the mass of that element as a percentage of the total molar mass are available in the "detail" box below;
The "chart" tab shows how the total molar mass of the compound is split between the individual elements making up the compound;
Alternatively enter your own chemical forumula in the "custom formula" box and press enter or click the calculate button.
from left to right;
copy the formula, as it appears, to the clipboard
copy the formula, in HTML format, to the clipboard
