Query Engine
The query engine is a quick and easy way of finding information about an element or set of elements. Rather than searching through the text or pages of the software you can issue a simple command here and the answer will appear immediately!
For those commands marked {name|symbol|atomic number} the name, symbol or atomic number of the element can be used.
Example: name Fe or name 26 would both return "iron".
Command legend;
utility
retrieves element properties
useful functions
The current version has the following commands;
help
Opens these help pages
clear
Clears the query screen
 
an {name|symbol}
Returns the atomic number of the requested element
am {name|symbol|atomic number}
Returns the atomic mass of the requested element
ar {name|symbol|atomic number}
Returns the atomic radius of the requested element
arb {name|symbol|atomic number}
Returns the bohr radius of the requested element
arc {name|symbol|atomic number}
Returns the covalent radius of the requested element
arv {name|symbol|atomic number}
Returns the van der Waals radius of the requested element
 
bp {name|symbol|atomic number}
Returns the boiling point of the requested element
 
dd {name|symbol|atomic number}
Returns the discovery date of the requested element
dy {name|symbol|atomic number}
Returns the density of the requested element
 
ec {name|symbol|atomic number}
Returns the electrical conductivity of the requested element
ehsell {name|symbol|atomic number}
Returns the electron shell configuration of the requested element
eof {name|symbol|atomic number}
Returns the energy of fusion of the requested element
eov {name|symbol|atomic number}
Returns the energy of vapourisation of the requested element
en {name|symbol|atomic number}
Returns the electronegativity of the requested element
 
hc {name|symbol|atomic number}
Returns the heat capacity of the requested element
 
ion {name|symbol|atomic number}
Returns the ion energy details for the requested element
 
mp {name|symbol|atomic number}
Returns the melting point of the requested element
 
name {symbol|atomic number}
Returns the name of the requested element
 
symbol {name|atomic number}
Returns the symbol of the requested element
 
tc {name|symbol|atomic number}
Returns the thermal conductivity of the requested element
te {name|symbol|atomic number}
Returns the thermal expansion of the requested element
 
ktoc {temperature in Kelvin}
Convert a temperate in Kelvin to Centigrade
ktof {temperature in Kelvin}
Convert a temperate in Kelvin to Fahrenheit
ctok {temperature in Centigrade}

ctof {temperature in Centigrade}
Convert a temperate in Centigrade to Fahrenheit
ftoc {temperature in Fahrenheit}
Convert a temperate in Fahrenheit to Centigrade
ftok {temperature in Fahrenheit}
Convert a temperate in Fahrenheit to Kelvin
