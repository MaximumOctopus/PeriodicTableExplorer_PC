Graphs
Clicking on a bar of the graph will open up the data page for the element that the bar represents.
The box in the centre of the graph settings panel displays the value (of the selected graph property) and the details of the element as the mouse moves across the graph display.
Clicking the "select property" button will bring up a menu that allows you so select which element property you wish to graph;
 
* Selecting ionisation by element or isotope abundance will open up the Element List drop down box (featured below), Select the element whose ionisation energies/abundance you want to view.
Choose which group of elements you want to view;
All (up to 103), the first 103 elements.

Alkali Earth Metals, Group 2

*Custom (the user selects which elements the graph will display)
This checkbox will only be available when either; Ionisation by element or isotope abundance is selected from the graph type, select which element's ionisation energies are display on the graph..
or custom is selected from the group selection box. Use to add an element to the graph, use to clear all elements from the graph.
Choose from Celsius or Kelvin temperature (where appropriate). Selecting the "non-zero" option will force the graph to omit entries where the value is zero. Selecting the "log" option will make the y-axis logarithmic.
Clicking the options button will bring up a list of menus that allow you to customise the look of the graph. There are also options for printing and saving the graph.
