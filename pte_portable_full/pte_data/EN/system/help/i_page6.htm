Atomic Radius Comparison
This application allows you to compare the relative atomic radii of different elements. Four different atomic radii are available: Atomic Radius, Bohr Atomic Radius, Covalent Radius and the van der Waals Radius. The radii are measured in picometres (pm). One picometre is equal to 10-12 (one trillionth) of a metre.
Each element's atomic radius is represented by a coloured disc. It is worth knowing that the boundaries of an atom aren't as clearly defined as this display would imply.
Clicking on the element name next to the coloured square that represents that element will open the relative Data Window.
