How to find....
Can't find that parameter or information? Then look no further! Click for instructions on how to find each item. Click here for terms commonly used throughout this page that reference the application.

Absorption spectrum
Can be found on the Periodic Display window, at the top.
 
Abundance
Main properties page in the data window, between the Sources and Uses sections.
