Data Window
The Data Window is where all of the data pages (biographies, glossary items, element data, etc.) are shown. There is no limit to the number of Data Windows that can be opened at any one time.
(a)
The arrows work the same as in most web browsers (Back/Forward), navigating the page history of the selected Data Window.
(b)
 Select from one of thirty tables which list the elements in order of a particular value.
(c)
 Opens the index of biographies.
(d)
 Opens the glossary index.
(e)
 This appears at the top of every element's data page. It supplements the initial information by displaying additional property pages for each element.

After clicking on an element, its data is displayed in this window. The background colour is linked to the selected element’s Group colour.
